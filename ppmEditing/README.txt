# Project: PPM file editing #
#############################

This is one of the course projects of EECS 402 of the University of Michigam. 
The purpose of the project is to enable to user to modify a ppm image. Three
functionalities are implemented in this project. 

1. Add a rectangle
2. Add a pattern
3. Add another PPM image

Sample files can be found in the project directory. 

A unix makefile is available for compling. No external library is needed.
For running the program, execute the .exe file and follow the instruction 
(a menu is provided in the program).
